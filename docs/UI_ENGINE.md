# EvoEngine UI Engine

## Verzió

v0.1.2

## Cél

A UI Engine célja, hogy a játék összes kezelőfelülete egységes, újrahasználható komponensekből épüljön.

## Komponensek

- UIElement
- UIManager
- UILabel
- UIPanel
- UIButton
- UIProgressBar
- UITheme

## Használat

A jelenetek saját UIManager példányt kapnak. Az elemek eseményt kezelnek, frissülnek és kirajzolódnak.

## Következő fejlesztések

- UISlider
- UICheckBox
- UIDropDown
- UITextBox
- ScrollView
- InventoryGrid
