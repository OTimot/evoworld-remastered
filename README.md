# EvoWorld Remastered

Open-world evolution survival game made with Python and Pygame.

## Version

v0.0.1 - Engine alap

## Indítás

```bash
pip install -r requirements.txt
python main.py
```

## Irányítás

- ENTER: játék indítása
- ESC: vissza / kilépés
- F11: teljes képernyő
- F3: debug mód
- WASD: játékos mozgatása a teszt jelenetben

## Projekt cél

A játékos egy egyszerű élőlényként indul, majd DNS pontokból fejlődik, alkalmazkodik a biomokhoz, harcol, gyűjtöget, evolválódik, és később multiplayer világban versenyezhet más játékosokkal.
